# SQL DDL and DML Foundations

This repository contains beginner-level SQL practice focused on Data Definition Language (DDL) and Data Manipulation Language (DML).

## Project Overview
The goal of this repository is to build a strong foundation in creating database objects, modifying table structures, applying constraints, and performing basic data operations such as insert, update, delete, and truncate.

## Topics Covered
- CREATE DATABASE
- CREATE TABLE
- ALTER TABLE
- RENAME TABLE
- TRUNCATE
- DROP
- PRIMARY KEY
- FOREIGN KEY
- UNIQUE
- CHECK constraints
- INSERT
- UPDATE
- DELETE

## Practice Scenarios
- Hospital database DDL exercises
- Online bookstore DML exercises

## Files Included
- `01_problem_statements/` - PDF question sheets
- `02_schema_and_setup/` - raw setup scripts
- `03_cleaned_solutions/` - cleaned final SQL solutions

## Key Learning Outcomes
- Understood the difference between DDL and DML
- Practiced table creation and structural changes
- Applied data integrity constraints
- Worked with insert, update, delete, and truncate operations
- Observed how foreign key constraints affect truncate behavior
